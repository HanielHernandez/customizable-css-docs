# Tables

## Basic table
To implement a basic style over tables you can use class **.table**

```html
<table class="table">
    <thead> 
        ...
    </thead> 
    <tbody> 
        ...
    </tbody> 
</table>
```


<table class="table mb-4">
  <thead>
      <tr>
          <th>Breakpoint</th>
          <th>Example</th>
      </tr>
  </thead>
  <tbody>
      <tr>
          <td>
              xl
          </td>
          <td>
              col-w-xl-3
          </td>
      </tr>
      <tr>
          <td>
              lg
          </td>
          <td>
              col-w-lg-4
          </td>
      </tr>
      <tr>
          <td>
              md
          </td>
          <td>
              col-w-md-6
          </td>
      </tr>
      <tr>
          <td>
              sm
          </td>
          <td>
              col-w-xl-12
          </td>
      </tr>
  </tbody>
</table>

## Hoverable table

You can add a hover effect by adding the class **.table-with-hover**

```html
<table class="table table-with-hover">
  ...
</table>
```

## Table bordered

<table class="table table-with-border">
