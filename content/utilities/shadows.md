# Shadows 

You can use shadows by adding the class **.shadow**

``` html
<div class="shadow-sm"> 
    ...
</div> 
<div class="shadow"> 
    ...
</div> 
<div class="shadow-lg"> 
    ...
</div> 
```

<div class="row mt-4">
                        <div class="p-4 shadow-sm mb-4 mr-4 r">
                                I'm a div with <b>.shadow-sm</b>
                        </div>
                        <div class="p-4 shadow mb-4 mr-3 r">
                            I'm a div with <b>.shadow</b>
