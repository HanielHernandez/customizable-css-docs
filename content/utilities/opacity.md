# Opacity

You can set opacity by using the class **opacity-{0 or multiple of 5}**
``` html
<div class="bg-blue-600 opacity-25 p-4">
    I have .opacity-25
</div>
<div class="bg-blue-600 opacity-50 p-4">
    I have .opacity-50
</div>
<div class="bg-blue-600 opacity-65 p-4">
    I have .opacity-65
</div>
```
